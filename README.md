## 记账本
数据不会被窃取的个人记账本。数据都存在本地，需要时，可以灵活导出。

## 功能
* 记一笔
  * 支出。选择类型，填写金额。
  * 收入。选择类型，填写金额。
* 概览
  * 时间维度。上月，本月，今日。默认是今天。
  * 盈余 `收入 - 支出`。
  * 支出
    * 总支出。
    * 按照分类显示支出。 用饼状图显示。
    * 如果时间维度是月，显示每天的折线图。
  * 收入
    * 总总支出。
    * 按照分类显示支出。 用饼状图显示。
    * 如果时间维度是月，显示每天的折线图。
* 导出
  * 导出收支数据。

说明，收支数据都存在 localStorage 中，因此不需要服务器。
